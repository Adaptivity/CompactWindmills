If you want to change the model, do the following steps:
1. Edit the model how you want it
2. Save the model into this folder
3. Export tthe model as Java file to compactwindmills_common/aroma1997/compactwindmills/rotors/ModelRotor.java
4. Fix the Package name
5. Fix the Imports
6. Add the Entity to the setRotationAngles
7. Change the second last parameter of addBox() to "16 * radius"
8. Change the parameter of ModelRotor() to int radius
Done!!
